Christmas Movie API - Documentation

This documentation outlines the use of the movie API which allows users to find information on a chosen movie, make it a favourite and manage their user account

Request URL HTTP Method Request body data format Response body data format
Return a list of all movies /movies GET None A JSON object holding data about all of the movies
Return information on a single movie requested by the user /movies/[TITLE] GET None A JSON object holding data about a single movie containing the name, genre, director, description, image URL and whether it’s featured or not
Return data about a director by name /movies/director/[DIRECTOR-NAME] GET None A JSON object holding a list of movies by a chosen director
Return data about a genre (description of genre) by name/title (e.g. Thriller) /movies/genre/[GENRE] GET None A description of a genre and a list of movies in that chosen genre
Allow new user to register /users POST A JSON object containing details about the new user with structure

{
username: “MarkHoppus”,
password: “xyz”,
email: “x@y.com",
dob: “18/10/86”
} 		A text message indicating whether the user was successfully added
Allow user to update their user info (username, password, email, DOB) /users/[USERNAME] PUT A JSON object containing the piece of info to update e.g.

{
username: “GeoffHoppus”
} 		A text message indicating whether the update was successful
Allow user to add a movie to their list of favourites /users/[USERNAME]/[FAVOURITE] PUT None A JSON object showing all of the users favourite movies including the new addition
Allow users to remove movie from their list of favourites /users/[USERNAME]/[FAVOURITE] DELETE None A text including whether the update was successful
Allow users to deregister users/[USERNAME] DELETE None A text message indicating whether the user was successfully deleted
